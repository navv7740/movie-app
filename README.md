# Movie App 

A movie search application built with HTML, CSS, and JavaScript. It allows users to search and explore movie information using an external movie API.

## Features

* Search movies by title
* Displays posters and details
* Responsive interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Movie API

## Setup

1. Replace:

const apiKey = "YOUR_API_KEY";

with your own API key.

2. Open "index.html".
